# Design: i18n with Paraglide.js (EN + Bahasa Indonesia)

**Date:** 2026-03-08
**Status:** Approved
**Scope:** All 57 SvelteKit pages, ~2000 translation keys

## Decision

Use **Paraglide.js 2.0** (@inlang/paraglide-js) for internationalization.

**Why:** Type-safe at compile time, tree-shakable (only used strings bundled per page), Svelte 5 native, cookie-based locale switching built-in, recommended by SvelteKit team, actively maintained.

**Languages:** English (en, base), Bahasa Indonesia (id)

**Switching:** Dropdown in navbar, cookie-based persistence, no URL prefix change.

## Architecture

```
front-end/
├── project.inlang/
│   └── settings.json          ← locales: ["en", "id"], baseLocale: "en"
├── messages/
│   ├── en.json                ← English strings (~2000 keys)
│   └── id.json                ← Bahasa Indonesia strings
├── src/
│   ├── app.html               ← <html lang="%lang%">
│   ├── hooks.server.ts        ← paraglideMiddleware (cookie strategy)
│   ├── lib/paraglide/         ← auto-generated by compiler (gitignored)
│   └── lib/components/ui/
│       └── LanguageSwitcher.svelte
├── vite.config.ts             ← paraglideVitePlugin
```

## Setup

### 1. Install
```bash
npx @inlang/paraglide-js@latest init
```

### 2. project.inlang/settings.json
```json
{
  "baseLocale": "en",
  "locales": ["en", "id"]
}
```

### 3. Vite plugin (vite.config.ts)
```typescript
import { paraglideVitePlugin } from '@inlang/paraglide-js';

export default defineConfig({
  plugins: [
    sveltekit(),
    paraglideVitePlugin({
      project: './project.inlang',
      outdir: './src/lib/paraglide',
      strategy: ['cookie', 'baseLocale'],
    })
  ]
});
```

### 4. Middleware (src/hooks.server.ts)
```typescript
import { paraglideMiddleware } from '$lib/paraglide/server';

export const handle: Handle = ({ event, resolve }) =>
  paraglideMiddleware(event.request, ({ request, locale }) => {
    event.request = request;
    return resolve(event, {
      transformPageChunk: ({ html }) => html.replace('%lang%', locale)
    });
  });
```

### 5. app.html
```html
<html lang="%lang%">
```

### 6. Usage in .svelte files
```svelte
<script>
  import * as m from '$lib/paraglide/messages';
</script>

<h1>{m.login_title()}</h1>
<p>{m.login_subtitle()}</p>
```

### 7. Language switcher
```svelte
<script>
  import { getLocale, setLocale } from '$lib/paraglide/runtime';
</script>

<select onchange={(e) => setLocale(e.target.value)}>
  <option value="en" selected={getLocale() === 'en'}>EN</option>
  <option value="id" selected={getLocale() === 'id'}>ID</option>
</select>
```

## Key Naming Convention

Flat keys, underscore-separated, namespaced by page/component:

```json
{
  "common_save": "Save",
  "common_cancel": "Cancel",
  "common_loading": "Loading...",
  "common_error": "Something went wrong",
  "login_title": "Sign in to your account",
  "login_email": "Email",
  "login_password": "Password",
  "dashboard_teacher_title": "Teacher Dashboard",
  "nav_teacher_dashboard": "Dashboard",
  "nav_teacher_students": "Students"
}
```
