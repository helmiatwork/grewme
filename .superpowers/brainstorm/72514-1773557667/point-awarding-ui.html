A. Student Grid (ClassDojo-style)

All students visible in a grid. Tap a student to open behavior selector popup. One tap to award. Great for quick classroom scanning.

Pros

  • Fast — see everyone at a glance
  • Familiar ClassDojo pattern
  • Works on projector

Cons

  • Gets crowded with 30+ students
  • Extra tap to select behavior

B. Behavior-First Quick Bar

Select the behavior FIRST, then tap multiple students to apply it. Great for "3 students helped today" scenarios. Batch-friendly.

Pros

  • Batch award to multiple students
  • Fewer total taps
  • Behavior always visible

Cons

  • Less intuitive initially
  • Reversed mental model

C. Both Modes (Tab Switch)

Grid view for individual awarding + Quick bar for batch awarding. Teacher picks which mode fits the moment.

Pros

  • Best of both worlds
  • Flexible for different situations

Cons

  • More complex to build
  • More UI to learn
