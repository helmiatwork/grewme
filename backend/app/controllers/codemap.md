# app/controllers/

## Responsibility
Minimal HTTP layer. In this GraphQL-first app, controllers are thin — the primary controller is GraphqlController which receives all API requests and delegates to the GraphQL schema.

## Design
- **ApplicationController**: API-only base class with `Authenticatable` concern
- **GraphqlController**: Single endpoint (`POST /graphql`). Skips CSRF. Extracts JWT from Authorization header, decodes token, finds user, passes context to schema.
- **Avo controllers**: Auto-generated by Avo gem for admin panel CRUD

## Flow
1. HTTP POST /graphql → GraphqlController#execute
2. Extract `Authorization: Bearer <token>` header
3. JWT.decode → find Teacher/Parent/SchoolManager by ID
4. `GrewmeSchema.execute(query, variables:, context: { current_user:, request: })`
5. Return JSON response

## Integration
- **Delegates to**: GrewmeSchema for all business logic
- **Uses**: Authenticatable concern for JWT decoding
- **Avo controllers**: Mounted at `/avo` for admin panel
