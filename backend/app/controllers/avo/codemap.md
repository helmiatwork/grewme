# app/controllers/avo/

## Responsibility
Avo admin panel controllers. Auto-generated by Avo gem for admin CRUD operations on all models.

## Design
- Inherits from Avo::ApplicationController
- Provides standard index/show/create/update/destroy actions
- Admin authentication handled by Avo session management

## Flow
Admin user accesses /avo → Avo routes to appropriate controller → Renders admin UI

## Integration
- **Wraps**: All ActiveRecord models via Avo resource definitions
- **Authentication**: Separate from JWT (Avo session-based)
